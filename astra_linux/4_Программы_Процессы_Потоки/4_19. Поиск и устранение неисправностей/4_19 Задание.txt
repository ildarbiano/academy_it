Задача:
1.	Как загрузить ОС в однопользовательском режиме (ответ в одну строку)
В меню GRUB нажимаем «e» (редактирование команд загрузки). 
Найти строку, начинающуюся с linux и в конце припишите параметр single, после чего нажмите Ctrl-x.


2.	Опишите последовательность восстановления работоспособности системы, если у вас ОС не грузится из-за ошибки в UID подключаемого диска.
! Нужен загрузочный диск или USB-накопитель и восстановить загрузчик.
Загрузите машину с "Live CD" - Live Astra Linux-дистрибутива.
Войдите в "Режим восстановления" - "recovery mode".
sudo fdisk -l : посмотреть таблицу разделов. раздел: /dev/sda1 * 
sudo mount /dev/sda1 /mnt : примонтировать раздел /dev/sda1
sudo chroot /mnt : переключиться в среду установленной ОС
sudo grub-install --recheck /dev/sda : переустановить GRUB 
sudo apt-get install linux-image : обновить ядро
exit : выйдите из chroot
sudo reboot : перезагрузить

UUID (Universally Unique Identifier) 	— это уникальный идентификатор, который присваивается каждому диску или разделу
В Linux
Проверить правильность UUID в файле /etc/fstab с помощью команды blkid для каждого устройства. Если UUID неверен, нужно исправить записи в файле.
Использовать live-образ системы для доступа к файлу /etc/fstab и редактирования UUID, если обновление initramfs не помогло.
Initramfs (initial RAM filesystem) 	— это небольшой корневой файловый системный образ, который загружается в оперативную память при запуске операционной системы Linux.
