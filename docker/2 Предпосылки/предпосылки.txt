-------
Virtual Machine
Hypervisor - OS по управлению другими ОС. Контролирует потребление ресурсов между VM.
Между ОС VM нет никакого доступа
- выбор ОС для пирложений небольшой Centos, Red Hat, Ubuntu
- развёрнуто несколько ОС, которая требует 10% ресурсов * Х количество VM
------
Container Runtime
Программный интерфейс управления в рамках одной ОС
Docker		- средство виртуализации рабочих сред на сервере. Запущенный процесс ОС в изолированном окружении с подключённой файловой системой из образа.
Docker_file	- манифест, где описан запуск контейнера	
Image		- исполняемый пакет, содержащий всё для запуска
