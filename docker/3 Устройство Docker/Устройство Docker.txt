Docker Compose	— это инструмент для управления многоконтейнерными приложениями на основе Docker. Простыми словами, он позволяет запускать сразу несколько Docker-контейнеров, настроенных для работы друг с другом, через один конфигурационный файл. В отличие от Docker, который работает с отдельными контейнерами, Docker Compose ориентирован на работу с группами контейнеров, объединённых в сервисы. Ими могут быть веб-серверы, базы данных, приложения и другие компоненты.
